This utilityPack contains various scripts from shaders to normal c# unity related content
Its content should be usable in most projects without the need of external code other that the one
of the unity's API.

It has various tools and in game functionalities such as: 
	- procedural generation 		(based on heightmaps or marching cubes, it uses shaders)
	- inventory system 			(modular, versatile and heavily customizable)
	- Utilities that holds expanded 
	  functionalities the unity API hasn't.
	- chunk system 				(still in experimental mode)
	- crafting system 			(still in development)
	- base first person controller		(limited)
